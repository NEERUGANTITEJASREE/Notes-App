# 📝 Java Notes App (File I/O)

## 📌 Description

A simple console-based Notes Manager built using Java. It allows users to write and read notes using file handling, ensuring data is saved permanently.

## 🎯 Features

✍️ Add notes to a file
📖 View saved notes
💾 Data persistence using file storage
🔁 Menu-driven program

## 🛠️ Technologies Used

Java (JDK)
FileWriter / FileReader / BufferedReader
VS Code / IntelliJ

## ▶️ How to Run

```bash
javac NotesApp.java
java NotesApp
```
